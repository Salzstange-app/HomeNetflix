
# Homeflix

U need something to get nzb files and something like Usenet

Its only work for Linux




## Run Locally

Create a Folder

```bash
  mkdir my-Homeflix
```

Go to the project directory

```bash
  cd my-Homeflix
```

Clone the project

```bash
  git clone https://github.com/Salzstange-app/Homeflix.git
```

Install dependencies

```bash
  bash .sh
```
